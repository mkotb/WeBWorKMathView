WeBWorK MathView

This Chrome extension will generate live previews of the equations you type into WeBWorK text boxes!

To enable this extension, perform the following steps:
  1. Open your WeBWorK website (for any one of your courses) in a separate tab and login if necessary
  2. Copy the URL from the address bar of Chrome
  3. Paste the full URL into the field below
  4. Press the checkmark to save your changes
  5. Select "Allow" if Chrome asks you to give permission to this extension to modify content on webpages under the URL you entered
  6. Refresh the WeBWorK page and your live previews should now appear
